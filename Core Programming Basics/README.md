Core Programming Projects

Overview:

This repository contains beginner-level Java projects developed while learning Core Programming concepts. These projects focus on programming fundamentals, problem-solving, debugging, and source code management using Git and GitHub.

Projects:

1\. Type Conversions Project:

A Java application demonstrating different types of data type conversions.

Concepts Used:

Variables and Literals

Primitive Data Types

Type Casting

User Input

Operators

Features:

Implicit Type Conversion

Explicit Type Conversion

Display conversion results



2\. OOPS Banner App:

A simple Java application that displays customized banners and messages.

Concepts Used:

Variables and Literals

Methods

String Manipulation

Conditional Statements

Features:

Display welcome messages

Display formatted banners



3\. Hello App

A beginner Java application for understanding basic program structure.

Concepts Used:

Java Program Structure

Variables

Basic Input and Output

Features:

Display greeting messages

Take user input

Print formatted output



4\. Guessing App

A number guessing game where users attempt to guess a randomly generated number.

Concepts Used:

Conditional Statements (if-else)

Loops (while)

Operators

User Input

Debugging

Features:

Random number generation

User guess validation

Hint messages

Multiple attempts until correct guess
