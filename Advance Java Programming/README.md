BookMyStayApp:

Overview:

BookMyStayApp is a simple Java-based hotel room booking application. 



It demonstrates core Java concepts including:

Classes and Objects

Collections Framework

Generics

Exception Handling

Regular Expressions

Custom Annotations

Reflection API

Sorting Collections



Features:

Book a hotel room for a guest

Validate guest email using Regular Expressions

Manage room inventory

Display booked guests in sorted order

Custom exception for unavailable rooms

Custom annotation for booking validation

Reflection-based annotation checking



Technologies Used:

Collections Framework

Regex

Annotations

Reflection API
