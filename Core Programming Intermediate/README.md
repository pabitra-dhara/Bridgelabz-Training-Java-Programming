Tic Tac Toe App

Overview:

The application uses a 3x3 game board, accepts player moves, validates input, checks winning conditions, and declares the game result.



Objectives

Understand Core Java building blocks.

Practice logical problem-solving.

Implement functional programming concepts.

Learn game state management using arrays.

Improve code organization through methods and modular design.

Features:

Two-player gameplay.

Interactive console interface.

Input validation.

Win condition detection.

Draw condition handling.

Board display after every move.

Technologies Used:

Array

Function

if-else
