Train Management App

Project Description:
This is a simple Java console application that manages train bogies.
The program stores bogie order using ArrayList and bogie information using HashMap.

Technologies Used:
ArrayList
HashMap
Enum

Enum is used to define:
TrainClass → SLEEPER, AC, FIRST
Shape → RECTANGULAR, CYLINDRICAL

Features:
Add multiple bogies using user input
Store bogie details
Display all bogie information
