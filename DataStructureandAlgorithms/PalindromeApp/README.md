Palindrome Checker App



Project Description:

This is a simple Java console application that checks whether a given string is a Palindrome or Not using:

Stack (LIFO - Last In First Out)

Queue (FIFO - First In First Out)



The program compares characters from both ends using stack and queue logic.





What is a Palindrome?

A Palindrome is a word, phrase, or sequence that reads the same forward and backward.

Examples:

madam,racecar



Features



1.Takes user input from console

2.Converts string to lowercase

3.Removes spaces

4.Uses Stack and Queue

5.Simple and beginner-friendly logic



Technologies Used



1.Stack (java.util.Stack)

2.Queue (java.util.Queue)
