BookMyStay App

Description:

BookMyStay is a simple Java console application that manages hotel room bookings.

The application allows users to check room availability and book rooms with optional add-ons like breakfast.



The project demonstrates the use of HashMap, PriorityQueue, and synchronized methods in Java.



Features:



View available room types

Book rooms easily

Prevent double booking using synchronized

Store room inventory using HashMap

Manage booking order using PriorityQueue



Optional add-ons:



Breakfast



Technologies Used



HashMap

PriorityQueue
