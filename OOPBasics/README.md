Employee Payroll App

Overview:

The Employee Payroll App is a console-based Java application that allows employees to securely log in and view their payroll information, including salary, tax deductions, other deductions, and net pay.

This project demonstrates Object-Oriented Programming (OOP) concepts such as classes, objects, constructors, access modifiers, methods, encapsulation, and authentication mechanisms.



Features:

Employee Login Authentication

Secure Username and Password Verification

View Salary Details

View Tax Information

View Deduction Details

Calculate Net Pay

Console-Based User Interface

Technology:

Classes, Variables and Methods

Constructors
