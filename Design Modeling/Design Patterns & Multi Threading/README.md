Trading \& Portfolio Tracking App:

This is a simple Java-based fintech simulation project that demonstrates:

Portfolio management

Asset pricing (Stock, Crypto)

Basic trading simulation

Price alerts system

Multithreading for trade execution

Features:

Create and manage portfolios

View asset prices using Factory Pattern

Send alerts to users using Observer Pattern

Execute trades using multithreading

Simple portfolio creation using Builder style

Singleton used for global portfolio management
