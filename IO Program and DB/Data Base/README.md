Health Clinic App

Overview:

Health Clinic App is a console-based Java application designed for a small clinic. It helps manage patients, doctors, appointments, and billing information using MySQL database connectivity through JDBC.

Features:

Register Patients

Manage Doctors and Specialties

Schedule Appointments

Track Visit History

Basic Billing Management

Store and Retrieve Data from MySQL Database

Technologies Used:

JDBC

MySQL Database

MySQL Connector/J Driver
