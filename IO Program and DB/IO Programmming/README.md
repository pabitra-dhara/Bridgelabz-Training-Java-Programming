IPL Analyzer App:

Overview:

This is a simple Java console application that works as an IPL data analyzer.  

The project demonstrates basic File Handling (CSV)

Features:

Add IPL match data using user input

Store data in CSV file

Read data from CSV file

Display match results in simple format

Easy and beginner-friendly implementation

Concepts Used:

Java File Handling

BufferedWriter \& BufferedReader

CSV Data Handling
